Options


theme
Type: String, Default: "default"

- Choose theme blue, green, yellow, red, black, purple and asphalt.


// Via JavaScript:
$(":file").jfilestyle({theme: "blue"});

// Via data attributes:
<input type="file" class="jfilestyle" data-theme="blue">

Example:







dragdrop
Type: Boolean, Default: true

- Enables or disables drag and drop selected files.


When enabling the drag and drop event multiple file selection is enabled. That is, you can not limit the amount of uploaded files in your upload using drag and drop. 
// Via JavaScript:
$(":file").jfilestyle({dragdrop: false});

// Via data attributes:
<input type="file" class="jfilestyle" data-dragdrop="false">

Example:

input
Type: Boolean, Default: true

- Enables or disables the input text.

// Via JavaScript:
$(":file").jfilestyle({input: false});

// Via data attributes:
<input type="file" class="jfilestyle" data-input="false">

Example:

text
Type: String, Default: "Choose file"

- Changes the button text.

// Via JavaScript:
$(":file").jfilestyle({text: "Find file"});

// Via data attributes:
<input type="file" class="jfilestyle" data-text="Find file">

inputSize
Type: String, Default: "200px"

- Change width of the input. Use units of measurement css.

// Via JavaScript:
$(":file").jfilestyle({inputSize: "400px"});

// Via data attributes:
<input type="file" class="jfilestyle" data-inputSize="400px">

Example:

placeholder
Type: String, Default: ""

- Set input placeholder.

// Via JavaScript:
$(":file").jfilestyle({placeholder: "My place holder"});

// Via data attributes:
<input type="file" class="jfilestyle" data-placeholder="My place holder">
